[Wolfgang Schwanghart](http://physiogeo.unibas.ch/staff/member.php?lang=en&memberID=6) from the Geographisches Institut, Universit√§t Basel has recently released a toolbox that allows analysis of relief and flow pathways in digital elevation models in Matlab. The toolbox can be used to visualize DEMs, extract simple derivatives, run (and modify) flow models, delineate drainage basins, produce hydrographs and implement similar DEM-based hyrdological analysis. For more info see the [User Guide to TopoToolbox](http://physiogeo.unibas.ch/topotoolbox/html/usersguide.html).

When you use TopoToolbox in your work, please refer to this publication:

Schwanghart, W., Kuhn, N. J. (2010): [**TopoToolbox: a set of Matlab functions for topographic analysis**](http://dx.doi.org/10.1016/j.envsoft.2009.12.002). Environmental Modelling & Software, in press.
